What is HTML5?

This site utilizes HTML5: a markup language used for structuring and presenting content on the World Wide Web. It's actually three kinds of code: HTML, which provides the structure; Cascading Style Sheets (CSS), which take care of presentation; and JavaScript, which makes things happen.
